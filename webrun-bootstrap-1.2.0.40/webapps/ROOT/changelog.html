  • CHANGE 11640[CORE] Implementao do parmetro avanado do Webrun SameSite, definido no grupo Cookie tem por objetivo previnir o envio do Cookie em requisies entre sites. Por padro o Webrun define o valor Lax, porm possvel definir os seguintes valores:
    • Lax: O cookie same-site ficar retido na sub-requisio entre sites, como chamadas para carregar imagens ou frames, mas ser enviada quando um usurio navegar para o URL de um site externo.
    • Strict: Se o cookie same-site possuir este atributo, o navegador enviar cookies apenas se a requisio for enviada do website que configurou este cookie, se a requisio tem origem em outra URL, nenhum cookie com o atributo Strict ser includo.
    Para mais informaes clique aqui.
  • CHANGE 11643[CORE] Implementao do parmetro avanado do Webrun DesabilitarScriptNaResposta, definida no grupo PropriedadesAvancadas quando ativo (valor defindo igual a 1) toda a resposta do servidor desabilitar a execuo de scripts.
    Observaes:
    1. Esse parmetro prioritrio no sistema, os demais parmetros que tem por objetivo a mesma funcionalidade sero sobrepostos.
    2. Caso o parmetro avanado VersaoSeguranca esteja ativo, por padro a propriedade se torna ativa.
    3. Esse parmetro s entra em vigor quando o sistema est publicado.
  • CHANGE 11642[CORE] Implementao do parmetro avanado do sistema "MaxAttemptsChangePassword". Esse parmetro tem como objetivo definir a quantidade mxima de tentativas que o usurio tem para redefinir a senha de acesso ao sistema, quando excede o limite a sesso do usurio expira e o mesmo redirecionado para a tela de login do sistema. Por padro, esse parmetro quando no definido pelo desenvolvedor assume o valor 5 (representando cinco tentativas).
